# PySpark-Sales-Data-Analysis
Overview

This project was created to practice Apache Spark and PySpark DataFrame operations using sample sales data. It focuses on learning the fundamentals of distributed data processing and performing common data analysis tasks with PySpark

Objectives

Load CSV files into PySpark DataFrames
Perform data exploration and cleaning
Practice filtering and selecting data
Group and aggregate data
Sort and analyze results
Build a strong foundation in PySpark
